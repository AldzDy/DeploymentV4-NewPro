How to use:

  1. Enter a unique site name
  2. Upload an HTML file or ZIP archive
  3. Click "Deploy to Vercel"
  4. Wait for deployment to complete
  5. Access your live site via the provided URL
Note: The system automatically rotates between 5 Vercel tokens if one fails.
